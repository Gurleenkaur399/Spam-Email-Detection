# Spam Email Detection using Machine Learning

## Overview
This project detects whether an email or SMS message is **Spam** or **Ham (Not Spam)** using Machine Learning.

## Features
- Data Cleaning
- Text Preprocessing
- CountVectorizer
- Multinomial Naive Bayes
- Model Evaluation
- Spam Prediction

## Dataset
- mail_data.csv

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
